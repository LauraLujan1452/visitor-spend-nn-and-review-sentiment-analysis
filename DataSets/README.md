# Data Sources

Raw data files are not committed to this repo due to file size limits.

* **international-visitors-london.csv** — included directly in this repo (see `DataSets/` folder).
* **shopee_reviews.csv** — [Shopee Text Reviews](https://www.kaggle.com/datasets/shymammoth/shopee-reviews) on Kaggle (137MB, not committed due to GitHub's file size limit). Download and place in this folder before running the notebook. Note: the notebook only uses the first 10,000 rows (`nrows=10000`).
